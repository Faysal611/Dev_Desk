Question-1: What are the different ways to select an element in the DOM?

DOM elements can be selected with getElementById to select with ID, getElementsByClass to select multiple elements with class. querySelector and querySelectorAll can also be used.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerText can select only the visible texts, whereas textContent is used to get all the text including hidden text. innerHTML gives us complete HTML element.

Question-3: What is event delegation in the DOM?

Event delegation is way of triggering an event using event bubble. Here event listener is applied only to parent and is triggered when target of an event is it's child.

Question-4: What is event bubbling in the DOM?

When a event is performed on a child this event propagates to the parents as well, this is known as event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, createElement is used. appendChild() is used for adding it to DOM. Lastly, remove() is used to removed any DOM element.
